# HR Analytics Project

## 📊 Project Overview
This project analyzes employee attrition using:
- Excel
- SQL
- Tableau
- Power BI

## 🔍 Key Insights
- Identified factors affecting employee attrition
- Department-wise attrition trends
- Salary and job role impact analysis

## 🛠️ Tools Used
- SQL (Data cleaning & querying)
- Tableau (Visualization)
- Power BI (Dashboard)

## 📁 Files Included
- hr_analysis.sql
- Tableau Dashboard (.twbx)
- Power BI Dashboard (.pbix)
- Presentation (.pptx)

## 📌 Conclusion
This project helps organizations understand and reduce employee attrition.
